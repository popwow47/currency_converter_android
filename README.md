## ✨ Возможности

- Конвертация между **фиатными** и **крипто** валютами в реальном времени
- Источники курсов: **CoinGecko** (крипто) + **ExchangeRate-API** (фиат)
- Выбор валют через удобные списки с флагами и иконками
- Кнопка обмена валют (swap)
- Индикатор последнего обновления курсов
- Ограничение запросов к API (счётчик в приложении)
- Проверка интернет-соединения
- Минимальная версия Android: **7.0 (Nougat)**


## Требования

- Python **3.9** или новее
- Flet **0.25.2** (или новее)
- Android: **7.0 (Nougat)**


**Как запустить:**
1. Скачайте `MY currency converter.apk`
2. Запустите файл
